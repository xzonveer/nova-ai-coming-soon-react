Nova AI - Coming Soon React Template

Version: 1.0.0 | Author: Your Name | Released: Oct 2025

Overview

A futuristic, responsive React landing page for AI agencies. Features particles animation, countdown timer, email signup, and feature teasers. Built with React 18, Tailwind CSS.

Requirements

  • Node.js 14+
  • npm 6+

Installation

  1. Extract source/nova-ai-coming-soon.
  2. Run npm install.
  3. Run npm start (dev server: http://localhost:3000).
  4. For production: npm run build â†’ Serve build/ folder.

Customization

Agency Name & Text

Edit src/App.js: Change <h1>Nova AI</h1> and hero <p>.

Launch Date

In src/components/Countdown.js: Update new Date('2025-12-01T00:00:00').

Features

Modify array in src/components/Features.js.

Email Integration

In src/components/SignupForm.js: Add EmailJS (npm install emailjs-com). Replace console.log with emailjs.sendForm. Get keys from emailjs.com.

Colors/Styles

Edit src/index.css for gradients/shadows. Tailwind config in tailwind.config.js.
